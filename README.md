# pm25-multimodal-prediction

PM2.5 Prediction using Multimodal Learning

This project develops a multimodal machine learning pipeline to predict PM2.5 levels by combining satellite imagery and structured environmental data.

Overview

Air pollution, particularly PM2.5, poses significant risks to public health. This project explores how integrating image-based information with structured environmental variables can improve prediction accuracy.

Method
Satellite image collection and preprocessing
Feature extraction from images
Integration with tabular weather/environmental data
Model training:
Baseline model (tabular only)
Multimodal model (image + tabular)
Project Structure
src/
├── data/
├── preprocessing/
├── models/
Future Work
Improve image feature extraction
